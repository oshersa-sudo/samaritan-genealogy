# שינויים שנוספו מהטקסטים הסרוקים — לעיון ואימות

כל אדם להלן **נוסף** לעץ מתוך הטקסט (התמלולים page_74–98 / הספר "תולדות בני ישראל השומרונים"), ולא היה במיפקד 1909 המקורי. השנים לועזיות. ההורה מצוין בסוגריים.

## ב. בית אב הדנפים


### משפחת הסתרי הדנפי (ישמעאל בן ישראל)

- **סעד בן ישמעאל** (1824–1882) — אב: ישמעאל
  - בן ישמעאל הסתרי, אחי ישראל ועבד-חנונה. משירה. נשא את זמהרה בת יעקב הצפרי. ילדיו: אבּ-ספֿוה, עבד-הרחום וקמרה.
- **אבּ-ספֿוה (אב-סכוה)** (1854–1912) — אב: סעד בן ישמעאל
  - בן סעד. היסטוריון ופקיד אצל סוחרי שכם; חיבר מילונים ערבית-עברית. נשא את אמינה בת אבּ-ספֿוה הדנפי. ילדיו: סעדיה, רסמיה ואסעד.
- **סעדיה** (1887–1950) — אב: אבּ-ספֿוה (אב-סכוה)
  - בת אבּ-ספֿוה. הראשונה שלמדה לקרוא ערבית מאביה. נישאה לפרץ בן אברהם הצפרי.
- **רסמיה** (1891–1972) — אב: אבּ-ספֿוה (אב-סכוה)
  - בת אבּ-ספֿוה. נישאה לאברהם בן יוסף בן עבד-חנונה הסתרי.
- **אסעד (סעד)** (1896–1974) — אב: אבּ-ספֿוה (אב-סכוה)
  - בן אבּ-ספֿוה. חנווני; חנות מכולת; עבד ביפו; בקיא בתרגום מילים מערבית לעברית מן המילונים שחיבר אביו. (אבי הענף הסראוי המודרני; השם נכתב גם "סעד".)
- **עבד-הרחום** (1856–1917) — אב: סעד בן ישמעאל
  - בן סעד. רוכל; מכר דברי סדקית בכפרים. נשא את מרים בת חביב הצפרי. ילדיו: נבון, לגזה, וגֿיה וצער.
- **נבון (פֿתֻמַי)** (1901–) — אב: עבד-הרחום
  - בן עבד-הרחום. עבד אצל סוחר בשכם, פתח חנות בדים, עבר לחולון; העתיק ספרים; ידע בראשית ושמות בעל-פה.
- **לֻגֿזה (רוזה)** (1902–1985) — אב: עבד-הרחום
  - בת עבד-הרחום. נישאה ליששכר בן אברהם המרחיבי; בעלת תפישה מהירה; בקיאה במסורות ובחשבון הימים.
- **וגֿ'יה (ע'גֿ'יה)** (1905–) — אב: עבד-הרחום
  - בן עבד-הרחום. רוכל בדים, פתח חנות בדים; בקיא בתפילה ובספרי בראשית ושמות; זקן צלותה.
- **צער (סֻפֿחִי)** (1908–1918) — אב: עבד-הרחום
  - בן עבד-הרחום. ילד פיקח; מת ממחלה בילדותו.
- **קמרה (ירחה)** (1859–1941) — אב: סעד בן ישמעאל
  - בת סעד. נישאה לאברהם מרחיב הצפרי; חיה עמו ביפו.
- **תמים (פֿאמל)** (1870–1949) — אב: עבד-חנונה
  - בן עבד-חנונה. אדם חכם; עסק בכתיבת עצומות ובקשות לבתי משפט; מנהל חשבונות; העתיק כתבי-יד רבים בעברית ובערבית. חי ביפו 1940–1947, שב לשכם. נשא את זינאב (פועה) בת אברהם.
- **פוֹזי** (1872–1923) — אב: תמים (פֿאמל)
  - בן תמים (מאשתו לאה). עזר לאביו בכתיבת בקשות; חלה בנמק ברגלו ומת בניתוח קטיעה.
- **סברי** (1900–) — אב: תמים (פֿאמל)
  - בן תמים. עסק בנגרות כל ימיו; העתיק כתב-יד או שניים.
- **ישמעאל (חיכמת)** (1904–1985) — אב: תמים (פֿאמל)
  - בן תמים. נקרא על שם אביו המת (נולד 4 ימים אחרי פטירתו). חייט; ידע על-פה בראשית ושמות; משירה; העתיק כתבי-יד רבים.
- **עבד-אלה (עבד אללטיף)** (1903–) — אב: אברהם
  - בן אברהם בן עבד-חנונה. חייט כל חייו; יצא לגמלאות והחל להעתיק ספרים.
- **סלים** — אב: אברהם
  - בן אברהם בן עבד-חנונה.

## ג. בית אב המרחיבים


### משפחת יהושע המרחיבי

- **קמרה (ירחה)** (1870–1950) — אב: חביב
  - בת חובב. נישאה לאפרים בן שת המרחיבי.
- **אברהם המרחיבי** (1886–1967) — אב: צפר (=סבאח)
  - בן צפר.

### משפחת מרחיב המרחיבי

- **אברהם (בן סעד)** (1846–1902) — אב: סעד
  - בנו בכורו של סעד. סופר ומעתיק בכתב יפה; פקיד אצל סוחרי שכם. נשא את סעדה ("אלטרמה") בת מרחיב, ואחריה את זפֿיקה בת בנימים המטרי.
- **זבולן (פיאד)** (1886–1967) — אב: אברהם (בן סעד)
  - בן אברהם. נגר; חי בשכם ועבר לחולון; מעריץ הכה"ג מצליח בן פינחס. נשא ב-1919 את פהֶימה בת צפר המרחיבי.
- **יעקב (בן סעד)** — אב: סעד
  - בן סעד (מאשתו אלפֻלֻה).
- **ברור (בן סעד)** — אב: סעד
  - בן סעד (מאשתו סעֻדֶה בת יהושע).
- **סֻעֻד (בן סעד)** (1868–1919) — אב: סעד
  - בן סעד; נקרא כך כי נולד אחרי מות אביו.
